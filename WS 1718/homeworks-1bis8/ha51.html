CSS Eigenschaften

Eigenschaft Erklärung mögliche Wertbelegung Beispiel
color sets the color of the selected content any CSS color unit color: red
font families setting a different font on the text Arial, Courier New, Georgia, Times New Roman, Trebuchet MS, Verdana font-family: times new roman
font size length and size unit of selected text px, em, rem font-size: 16px;
font style used to turn italic text on and off normal, italic, oblique font-style: italic
font weigth sets how bold the text is normal, bold, lighter, bolder, 100-900 font-weigth: bold
text transform allows you to set your font to be transformed none, uppercase, lowercase, capitalize, full-width text-transform: capitalize
text decoration sets/unsets text decorations on font none, underline, overline, line-through text-decoration: none
text shadows apply drop shadows to the selected text horizontal vertical blur text-shadow: 4px 4px 5px red
text alignment control how text is aligned within its containing content box right, left, center text-align: left
line-height sets the height of each line of text size line-height: 1.5
letter and word spacing set the spacing between letters and words in the text length and size units letter-spacing: 2px;
word-spacing: 4px;
web fonts alternative fonts that fit your website better anything from Google Fonts font-family: "myFont";
src: url("myFont.ttf");
